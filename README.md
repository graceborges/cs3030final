	Sub - eval_winner()
		What it does:
			This sub will determine the winner of the rock, paper,
			scissors game.

		Vars: 
			$p1 - the player's choice entered
			$p2 - randomly computed choice of CPU

		Returns:
			returns 0 - game results in a tie
			returns 1 - $p1 wins
			returns 2 - $p2 wins
